💼✨ Job Tracker App

A simple, and powerful app to help you keep track of your job applications without losing your mind 😌

🌸 Features
🔐 Authentication (powered by Better Auth)
📋 Track job applications in one place
🧠 Stay organized with your job search
⚡ Fast and modern UI (Next.js + Tailwind)
🎨 Clean & minimal design

🛠️ Tech Stack
⚛️ Next.js 16
🧩 React 19
🎨 Tailwind CSS
🔐 Better Auth
🍃 MongoDB
🧱 shadcn/ui

💖 How it works
Sign up / log in with email & password
Your personal job board is created automatically
Track your applications and stay organized
